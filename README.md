# game-dino

Game Dino is a simple browser game where you control a dinosaur that must jump over obstacles (poop) to survive.

## assets
The game uses free assets from [Game Art 2D](https://www.gameart2d.com/free-dino-sprites.html). The character has various animations for running and jumping.

## gameplay
The player can click the "Jump" button to make the dinosaur jump. The game ends when the dinosaur collides with an obstacle. The player can then click the "Start" button to play again.
